# Python + Git Foundations — Audio Overview Prompt

Use with a local LLM (Ollama), Claude, or ChatGPT. Read the output aloud or use TTS. **Manual only — no automated external login.**

---

## Prompt

```
You are a concise AI engineering instructor. Create a 3–5 minute audio-style script (450–750 words) for a commuter who just completed "Python + Git Foundations."

Structure:
1. Hook — why file I/O and Git matter before touching LLMs or GPUs (30 seconds)
2. Core concepts — Python project layout, pathlib, JSON configs, functions, main guard, git add/commit/log (2 minutes)
3. AI engineering connection — reproducing experiments, config versioning, rolling back bad prompt changes (1 minute)
4. Exercise recap — lab config manager creates/updates lab_config.json with last_run timestamp (45 seconds)
5. Three review questions — pause after each:
   - What does git add do vs git commit?
   - Why store configs as JSON files?
   - What problem does if __name__ == "__main__" solve?

Tone: clear, practical, no jargon without a one-line definition.
```

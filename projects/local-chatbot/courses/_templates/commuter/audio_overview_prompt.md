# [Lesson Title] — Audio Overview Prompt

Use this prompt with a local LLM (Ollama), Claude, or ChatGPT to generate a **3–5 minute spoken-style summary** before or after your commute. Read the output aloud or paste into a TTS tool.

**Do not automate external services.** Generate text locally or in your own chat session.

---

## Prompt

```
You are a concise AI engineering instructor. Create a 3–5 minute audio-style script (about 450–750 words) for a commuter who just completed the lesson "[Lesson Title]".

Structure:
1. Hook — why this matters for AI engineers (30 seconds)
2. Core concepts — explain each in plain language (2 minutes)
3. AI engineering connection — one concrete example (1 minute)
4. Exercise recap — what they built and why (45 seconds)
5. Three review questions — pause after each for the listener to answer

Tone: clear, energetic, no jargon without definition. Do not mention login, automation, or tools the listener has not installed yet.
```

## After Listening

Open `review_questions.md` and answer each question without notes.

# Linear Algebra Foundations for AI Engineering

## Lesson Goal
Understand vectors, dot products, cosine similarity, and why they matter for embeddings and LLM systems.

## Core Concepts
- Scalars
- Vectors
- Matrices
- Dot products
- Magnitude
- Cosine similarity
- Embeddings

## AI Connection
Vectors are how AI systems represent meaning. Embeddings convert words, documents, images, and code into vectors so machines can compare similarity.

## Build Task
Create a Python script that computes cosine similarity between two vectors.

## Checkpoint
Explain why embeddings are vectors and how similarity search uses linear algebra.
